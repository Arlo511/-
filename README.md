# 光影与留白 —— 摄影作品展示网站

## 项目简介

「光影与留白」是一个精致、现代的摄影作品展示网站，采用极简主义设计风格，旨在为摄影师提供一个优雅的作品展示平台。网站以大量的留白、精致的排版和丰富的动画交互效果为特点，让每一位访问者都能沉浸在摄影作品的艺术氛围中。

本项目采用纯前端技术栈构建，无需复杂的后端配置即可快速部署和使用。无论是专业摄影师展示作品集，还是摄影爱好者分享创作，这个网站都能满足您的需求。

## 主要特性

### 设计理念

本网站秉持「内容即核心」的设计理念，通过精心设计的视觉元素和交互体验，让观众的注意力完全聚焦于摄影作品本身。我们摒弃了繁杂的装饰元素，采用经典的黑白灰单色调作为主视觉，确保不会干扰照片本身的色彩表现。网站整体呈现出一种简约而不简单的高级感，每一个细节都经过反复推敲，力求在现代感与经典美学之间取得完美平衡。

### 核心功能

网站提供了完善的作品展示功能，包括全屏沉浸式的首页 Hero 区域，让访客在第一时间被精美的摄影作品所吸引。作品画廊支持多种分类筛选，包括风光、人像、城市和抽象四大类别，方便访客快速找到感兴趣的作品类型。每个作品都支持点击查看大图，灯箱弹窗功能提供了沉浸式的浏览体验，支持键盘导航和触摸滑动手势。

关于摄影师页面展示了摄影师的个人简介和创作理念，配合优雅的统计数据动画，让访客对摄影师有更深入的了解。联系页面提供了功能完整的表单系统，访客可以直接发送合作咨询邮件。网站还配备了社交媒体链接区域，方便建立更广泛的粉丝社区。

### 动画与交互

网站在交互体验上投入了大量心血，力求为每一位访客带来流畅优雅的浏览体验。自定义光标效果让鼠标操作更加精致有趣，光标会跟随鼠标移动，悬停在可点击元素上时会产生放大效果。页面加载动画精心设计，Logo 旋转淡入的效果既优雅又不会让访客感到等待的烦躁。

平滑滚动功能由 Lenis 库提供支持，带来如丝般顺滑的滚动体验，彻底告别浏览器原生滚动的生硬感。滚动触发的元素显现动画让页面内容随着滚动自然浮现，营造出层次分明的视觉体验。作品卡片支持悬停缩放和微妙的视差效果，鼠标移动时图片会产生跟随移动的效果，增加画面的立体感和互动性。

### 响应式设计

网站采用完整的响应式设计策略，在桌面端、平板和手机等各种设备上都能提供最佳浏览体验。桌面端采用多栏网格布局，充分利用大屏幕的优势展示更多作品内容。平板设备自动调整为双栏布局，保证内容可读性的同时优化空间利用。手机端采用单栏流式布局，配合专门设计的移动端导航菜单和触摸友好的交互方式，确保在任何设备上都能获得出色的使用体验。

## 快速开始

### 本地预览

要在本地计算机上预览网站，您只需要在项目根目录下使用任意静态文件服务器打开 `index.html` 文件即可。如果您安装了 Python，可以直接在项目目录中运行以下命令启动本地服务器：

```bash
cd photography-portfolio
python3 -m http.server 8080
```

然后在浏览器中访问 `http://localhost:8080` 即可查看网站效果。如果您使用 VS Code 编辑器，也可以安装「Live Server」扩展，然后右键点击 `index.html` 文件选择「Open with Live Server」来启动预览。

### 在线部署

网站支持多种静态网站托管服务的部署方式，包括 Vercel、Netlify、GitHub Pages 以及传统的虚拟主机部署。部署时只需将 `dist` 目录中的所有文件上传到您的托管空间即可。对于 Vercel 和 Netlify 等服务，您可以直接连接 GitHub 仓库进行自动化部署。

## 自定义指南

### 添加作品图片

要向网站添加新的摄影作品，请按照以下步骤操作。首先，将您的作品图片放入 `images/` 目录中，建议使用 WebP 格式以获得最佳的加载速度与质量平衡，如果无法生成 WebP，使用高质量 JPG 也是不错的选择。其次，在 `index.html` 文件中找到作品画廊区域，参考现有作品的代码结构添加新的作品条目。

以下是新增作品的标准代码模板：

```html
<article class="gallery-item" data-category="landscape">
    <div class="item-image-wrapper">
        <img src="images/your-image-name.webp" alt="作品标题" class="item-image" loading="lazy">
        <div class="item-overlay">
            <div class="overlay-content">
                <h3 class="item-title">作品标题</h3>
                <p class="item-category">风光摄影</p>
                <span class="view-btn">查看作品</span>
            </div>
        </div>
    </div>
</article>
```

在上面的代码中，请根据实际情况修改以下属性：`data-category` 设置为对应的分类（landscape 风光、portrait 人像、urban 城市、abstract 抽象）；`src` 属性指向您的图片路径；`alt` 属性填写图片的描述文字；`.item-title` 填写作品标题；`.item-category` 填写分类显示名称。

### 修改文字内容

网站中的所有文字内容都可以在 `index.html` 文件中直接修改。以下是主要区域的文字内容位置：首页标语位于 Hero 区域的 `<h1>` 和 `<p class="hero-subtitle">` 标签中；关于摄影师介绍位于 `<section id="about">` 区域内；联系信息和表单标签位于 `<section id="contact">` 区域内。修改后保存文件并刷新浏览器即可看到更新效果。

### 自定义颜色方案

网站的所有颜色变量都集中在 `css/style.css` 文件的 `:root` 选择器中。主要颜色变量包括：`--color-white` 控制背景色，默认为白色；`--color-black` 控制主要文字颜色，默认为近黑色；`--color-gray-500` 控制次要文字颜色，默认为灰色。修改这些变量的值即可快速调整网站的整体配色风格。

### 自定义字体

网站使用 Google Fonts 提供的字体服务，标题字体为 Cormorant Garamond，正文字体为 Inter。如果您需要更换字体，可以在 `index.html` 文件的 `<head>` 部分找到 Google Fonts 链接，将其替换为其他字体组合。修改后还需要在 `css/style.css` 中更新 `--font-display` 和 `--font-body` 变量的值。

### 添加新分类

要在网站中添加新的作品分类，需要进行两处修改。第一处是在筛选按钮区域添加新的按钮元素：

```html
<button class="filter-btn" data-filter="new-category">新分类名</button>
```

第二处修改是在对应作品的 `<article>` 标签中设置 `data-category` 属性的值与按钮中一致。

## 图片优化建议

为了确保网站始终保持流畅的加载速度，建议您遵循以下图片优化标准。作品图片的桌面端宽度建议为 1920 像素，这样可以在大屏幕上获得清晰的展示效果；移动端宽度建议为 800 像素，可以显著加快移动设备的加载速度。图片文件大小建议控制在 200KB 以内，过大的图片会延长加载时间，影响用户体验。

推荐使用 WebP 格式作为首选，这种格式在保持高质量的同时能够显著减小文件体积。如果您的图片编辑软件不支持 WebP，可以使用 Squoosh 等在线工具进行格式转换。对于必须使用 JPG 的情况，建议将质量设置在 80% 到 90% 之间，并在保存时移除 EXIF 数据以进一步减小文件体积。

## 技术栈说明

本项目采用纯前端技术栈构建，各技术组件的用途说明如下。HTML5 语义化标签确保代码具有良好的可读性和 SEO 友好性。CSS3 提供了丰富的样式能力，包括 Flexbox 和 Grid 布局、动画和过渡效果等。Vanilla JavaScript 处理所有用户交互逻辑，无需依赖笨重的框架。

GSAP（GreenSock Animation Platform）是业界领先的动画库，用于创建复杂的序列动画和滚动触发的效果。Lenis 提供了极致流畅的滚动体验，增加了惯性阻尼感，消除了浏览器原生滚动的生硬感。Google Fonts 提供高品质的网页字体服务，确保文字在不同设备上都能呈现最佳效果。

## 浏览器支持

网站支持所有现代浏览器的最新版本，包括 Chrome、Firefox、Safari 和 Edge。对于 Chrome，我们建议使用 80 以上版本以获得最佳的动画性能和滚动体验。Safari 用户建议使用 14 以上版本。移动端方面，iOS Safari 和 Android Chrome 都能提供完整的功能支持。

Internet Explorer 不受支持，因为该浏览器已经停止更新，且无法正确渲染现代 CSS 和 JavaScript 特性。如果您需要支持旧版浏览器，可能需要进行额外的兼容性处理。

## 常见问题

**问：如何修改网站的 Logo？**

网站 Logo 由 SVG 代码直接编写，位于 `index.html` 中导航栏和页脚的 `<svg>` 标签内。如果您需要更换为自定义 Logo，可以准备一个 SVG 格式的 Logo 文件，复制其 `<svg>` 标签内容替换现有代码即可。

**问：联系表单能收到邮件吗？**

当前的联系表单只提供了前端界面设计，要实现邮件发送功能，您需要连接后端服务或使用第三方表单服务。推荐的服务包括 Formspree、Netlify Forms 等，它们可以轻松地将表单提交转换为邮件通知，无需编写后端代码。

**问：如何添加更多的统计数据？**

统计数据位于关于部分的 `.about-stats` 容器中。添加新统计项的步骤如下：首先在 HTML 中添加新的 `<div class="stat-item">` 元素；然后设置 `<span class="stat-number">` 的 `data-count` 属性为您期望的最终数字；JavaScript 会自动为这些数字添加计数动画效果。

**问：作品图片加载很慢怎么办？**

首先检查您的图片文件大小，确保每张图片都在 200KB 以下。其次考虑使用 WebP 格式替代 JPG 或 PNG。第三可以使用图片懒加载功能，当前已通过 `loading="lazy"` 属性启用。最后可以考虑使用 CDN（内容分发网络）来加速图片加载。

**问：如何禁用自定义光标效果？**

如果需要禁用自定义光标效果，可以在 `css/style.css` 文件末尾添加以下代码：

```css
.custom-cursor,
.cursor-follower {
    display: none;
}

body {
    cursor: auto;
}
```
